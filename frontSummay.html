HTML and CSS Summary

HTML Summary

HTML(Hyper Text Markup Language) is used to write the actual content,you don't use it for styling purpose. use css instead!

  • HTMl uses "elements" to describe (annotate) content
  • HTML typically have an opening tag, content and then a closing tag
  • you also can have void elements like images
  • there is long list of available elements but you will gail experience over a time , no worry

Learn more all about available HTML elements the MDN HTML elements reference

CSS Summary

CSS (cascading stylesheet) is used for styling you page content

  • Styles are assigned via property value pairs
  • you can assign styles via the "style" attribute
  • to avoid code duplication , you typically use global styles
  • Alternatively, you can work with external stylesheet files which you "link"
  • when working with css inheritance, "specificity" and box model should be understood

Learn more all about available css properties onthe MDN css property reference
